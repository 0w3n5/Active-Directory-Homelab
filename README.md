# Active-Directory-Homelab
A virtual machine homelab for learning Active Directory attacks and security.

## Setup
The lab was made in virtual-box using a Windows 2019 server, 2 windows 10 machines and a Kali machine. The machines and server are connected with a NAT adapter, this is to keep this network isolated so I can deliberately make the machines and server vulnerable without having to worry about external threats. The 2 machines each have a user who is a local admin, one of the users is also an admin across all Windows machines.
